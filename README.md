# mloader
   移动端异步列表加载插件
   依赖jquery.js
   
###### 使用场景：

   在移动web端开发时，会做很多列表，为了更好的用户体验，你需要异步加载分页，如滚动到底部时能自动触发加载下一页。
    
   列表回退还原特性，这是一个用户体验极好的特性。列表页通常需要点击进入详情页面，如进入商品详情或新闻内容页后，用户习惯性地点返回键，页面会回退到列表页，这时列表页可以无缝还原列表页离开时的状态，无需让用户从头翻页下去。
   
   列表加载时需要灵活使用各种过滤参数。

###### 特性：

   列表内容返回后可还原列表页状态。 
   比如商品列表，新闻列表进入内容页后，回退到列表页时可还原列表的操作状态。 
   灵活支持各种过滤参数。 
   一个页面支持多个实例。
